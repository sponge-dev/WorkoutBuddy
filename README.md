# WorkoutBot - AI-Powered Fitness Coach

A comprehensive fitness application that uses OpenAI's GPT-4 to create personalized workout plans, track progress, and help you achieve your fitness goals. Built with Flask and optimized for both desktop and mobile use.

## 🚀 Features

### 🤖 AI-Powered Workout Plans
- **Personalized Plans**: Get custom workout plans generated by GPT-4 based on your goals, fitness level, and available equipment
- **Smart Recommendations**: AI considers your progress, preferences, and physical constraints
- **Progressive Training**: Plans automatically adjust as you progress

### 📊 Comprehensive Progress Tracking
- **Weight & Body Composition**: Track weight, body fat percentage, and muscle mass
- **Body Measurements**: Monitor chest, waist, hips, arms, and thigh circumferences
- **Visual Progress**: Beautiful charts and graphs to visualize your journey
- **BMI Calculator**: Built-in BMI calculator with health category indicators

### 📱 Mobile-Optimized Interface
- **Responsive Design**: Works perfectly on iPhone, Android, and desktop
- **Touch-Friendly**: Optimized for mobile interaction
- **Offline-Ready**: PWA capabilities for app-like experience

### 🎯 Goal-Oriented Training
- **Multiple Goal Types**: Bulk, cut, tone, strength, and endurance training
- **Equipment Flexibility**: Plans adapt to your available equipment
- **Progress Monitoring**: Track goal achievement with visual indicators

### 📈 Advanced Analytics
- **Workout Statistics**: Total workouts, training time, and frequency analysis
- **Progress Trends**: Identify patterns in your fitness journey
- **Performance Insights**: Data-driven recommendations for improvement

## 🛠 Technology Stack

- **Backend**: Python Flask with SQLAlchemy ORM
- **Database**: SQLite for easy setup and portability
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **UI Framework**: Bootstrap 5 for responsive design
- **Charts**: Chart.js for beautiful data visualization
- **AI Integration**: OpenAI GPT-4 API for workout plan generation
- **Icons**: Font Awesome for modern iconography

## 📋 Prerequisites

- Python 3.8 or higher
- OpenAI API key (already configured in `api_keys.json`)
- Modern web browser (Chrome, Firefox, Safari, Edge)

## 🚀 Quick Start

### 1. Install Dependencies

```bash
# Install Python dependencies
pip install -r requirements.txt
```

### 2. Run the Application

```bash
# Start the Flask development server
python app.py
```

### 3. Access the Application

Open your web browser and navigate to:
```
http://localhost:5000
```

### 4. Set Up Your Profile

1. Click "Get Started" on the home page
2. Fill in your personal information (name, age, height, gender, fitness level)
3. Set your fitness goals and preferences
4. Start generating AI-powered workout plans!

## 📱 Mobile Access

### iPhone/iPad Access
1. Open Safari and go to `http://your-computer-ip:5000`
2. Tap the Share button
3. Select "Add to Home Screen" for app-like experience

### Android Access
1. Open Chrome and go to `http://your-computer-ip:5000`
2. Tap the menu (⋮) and select "Add to Home screen"

## 🐛 Troubleshooting

### Common Issues

#### Application Won't Start
```bash
# Check Python version
python --version

# Reinstall dependencies
pip install -r requirements.txt --force-reinstall
```

#### Database Errors
```bash
# Delete and recreate database
rm workoutbot.db
python app.py
```

#### OpenAI API Issues
- Verify your API key in `api_keys.json`
- Check your OpenAI account balance
- Ensure stable internet connection

### Mobile Access Issues
- Use your computer's IP address instead of localhost
- Ensure both devices are on the same network
- Check firewall settings on your computer
